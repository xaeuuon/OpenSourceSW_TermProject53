# Privacy Protector (OpenCV)

## 1. 개요
웹캠을 통해 실시간으로 얼굴을 인식하고, 자동으로 모자이크 처리하여 프라이버시를 보호하는 프로그램.

## 2. 사용 패키지 및 버전
* Python 3.x
* opencv-python (4.8.0 이상)

## 3. 실행 방법
1. 라이브러리 설치: `pip install opencv-python`
2. 실행: `python main.py`
3. 종료: `q` 키 누르기
